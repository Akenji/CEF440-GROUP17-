## ğŸ“Œ Project Overview

This project is a **mobile-based attendance management system** designed to automate and secure student attendance in academic institutions using a combination of **facial recognition** and **geofencing** technologies.

Traditional attendance methods are often inefficient, error-prone, and susceptible to manipulation (e.g., proxy attendance). This system addresses those limitations by verifying both **identity (via face recognition)** and **physical presence (via GPS coordinates)**, ensuring accurate and real-time attendance monitoring.
